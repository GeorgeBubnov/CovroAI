Main module - Главный исполнимый модуль

main.py

Главный модуль приложения. Управляет взаимодействием всех модулей, инициализирует API и предоставляет интерфейс для пользователя.

main.main()

Основная точка входа в приложение.

main.run_analysis(url)

Выполняет анализ сайта и формирует рекомендации.

Параметры:

url – URL анализируемого сайта.

main.start_services()

Инициализирует все модули системы.
